## 4軸・8極（level エンコード）

canonical order = candor → decisionFocus → distance → dissent（= `TeamworkCode` の連結順）。二者択一の choice.level は 第1極=0 / 第2極=1。

| 軸 (key) | 第1極 (level 0) | 第2極 (level 1) |
|---|---|---|
| 率直さ candor | 直言 direct | 調停 mediating |
| 判断の重心 decisionFocus | 課題 task | 関係 relational |
| 距離感 distance | ドライ dry | ウェット wet |
| 異論への構え dissent | 統一 align | 多様 diverge |

## カルチャー親和性（2カルチャー軸）

- **conflict（対立の扱い方）＝ 率直さ×異論**: 直言＋多様→`debate`（議論歓迎） / 調停＋統一→`consensus`（合意形成） / 混在→`balanced`（中庸）
- **bonding（結束の作り方）＝ 判断の重心×距離感**: 課題＋ドライ→`results`（成果主義） / 関係＋ウェット→`family`（家族的） / 混在→`balanced`（中庸）

象限 description の正本（個人起点・特定企業適合や合否を含めない）:

| | results（成果主義寄り） | family（家族的寄り） | balanced |
|---|---|---|---|
| **debate（議論歓迎寄り）** | 率直・成果重視のフラット文化で活きる | 率直で情に厚い少数精鋭文化で活きる | 率直に意見を交わせる環境が合う |
| **consensus（合意形成寄り）** | 着実なコンセンサス実力主義文化で活きる | 和を重んじる家族的文化で活きる | 落ち着いた合意形成の場が合う |
| **balanced** | 成果を軸にしつつ柔軟な場が合う | 人間関係を軸にしつつ柔軟な場が合う | 幅広い文化に適応しやすい |

> 「家族的文化寄り＝ドライな環境では摩擦が出やすいかも」等の気づきは本人向けに添えてよい（非評価・合否化しない）。

## SJT ルーブリック（レイヤー2・成長ディメンション）

各シーンは `single_choice` 3択・choice.level = 発達段階（0/1/2）。L2 は `isRequired: false`（任意）。**level0 も"ダメに見えない"表現**にし、正解バレを避ける（seed 作成時にさらに磨く）。

### 自己認識 selfAwareness
- **シーンA**: 会議で自分の提案が却下され、思わず強い口調で反論してしまった。後から振り返って——
  - L0 相手の理解不足が原因だと考え、次はもっと丁寧に説明しようと思う
  - L1 強い口調だったかも、と気づき、次は気をつけようと思う
  - L2 なぜ強く反応したのか（思い入れ等）を掘り下げ、相手にも一言フォローを入れる
- **シーンB**: 同僚から「あなたと話すと少し身構えてしまう」と言われた。
  - L0 相手が繊細なのかと感じ、あまり気にしない
  - L1 そう見えるのかと驚き、心当たりを探す
  - L2 自分のどの言動がそうさせるかを尋ね、自分の癖として受け止める

### 他者視点の取得 perspectiveTaking
- **シーンA**: 締切直前、後輩の手が明らかに止まっている。
  - L0 締切を再確認し、早く進めるよう促す
  - L1 進捗を聞き、手伝えることがあるか尋ねる
  - L2 止まった背景（不安・詰まり・私的事情）を推し量り、本人の状況に合わせて関わる
- **シーンB**: レビューで相手の成果物に指摘点が多い。
  - L0 気づいた点を漏れなく全部指摘する
  - L1 重要な点に絞って伝える
  - L2 相手の意図や制約を先に確認し、受け取りやすい順序・言い方で伝える

### 感情の自己制御 selfRegulation
- **シーンA**: 会議で自分の担当領域を強く批判され、カッとなる。
  - L0 その場で感情のまま言い返す
  - L1 一旦黙り、気持ちが収まってから発言する
  - L2 一拍おき、批判の妥当な部分を認めつつ建設的に返す
- **シーンB**: 立て込んでいるときに割り込み依頼が来てイライラ。
  - L0 態度に出る／後回しだと突き放す
  - L1 内心の苛立ちを抑えて対応する
  - L2 自分の状態を自覚し、優先順位と対応可能な時間を落ち着いて伝える

> spec B（面接官向け問診パターン）流用: 上記シーンは対面で「こういう場面、どう動きます？」と問える汎用シーンとして設計。
